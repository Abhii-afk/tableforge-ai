# tableforge-ai

The Institutional Memory Layer for Your Database.

---

## Frontend

Initial setup using Next.js 14, Tailwind CSS, and shadcn/ui.
Includes the dashboard layout and core MVP components:
- Natural Language Query Bar
- Live Grid View
- Relationship Visualizer

### Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
